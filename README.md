# 🚀 GameVault | Professional MERN E-Commerce

GameVault is a high-performance web application built for hardware enthusiasts. It features a full-stack architecture, secure Stripe payments, and real-time inventory tracking.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Grid/Flexbox), JavaScript (ES6+)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Local Server
- **Payments:** Stripe API Integration

## 📦 Key Features
- **Dynamic Product Grid:** Filterable hardware collection with real-time stock badges.
- **Secure Checkout:** PCI-compliant transaction flow using Stripe Hosted Checkout.
- **Order Reconciliation:** Automated inventory decrementing upon successful payment.
- **Admin Dashboard:** Real-time revenue tracking and order history logging.
